歡迎使用 PlanX

輸入您的任務與需求天數，按下產生即可得到一週行程建議。您也可以編輯、標示完成以及儲存行程。

行程表

使用說明

  1. 在上方輸入您想安排的任務內容。
  2. 設定需要規劃的天數 (1-7)。
  3. 輸入有效的 OpenAI API Key 後點擊「產生行程表」。
  4. 每一天的任務可自由編輯，拖曳調整順序，或點選勾選框標示完成。
  5. 可透過新增任務按鈕在任意一天增加更多工作項目。
  6. 完成後可點擊「匯出行程」將行程存成 JSON 下載。
